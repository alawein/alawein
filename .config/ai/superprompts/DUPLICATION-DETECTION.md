# Duplication Detection Prompt

## Objective
Identify and analyze code, configuration, and workflow duplication across the monorepo.

## Analysis Framework

### Code Duplication
- Duplicate functions/classes
- Similar logic patterns
- Copy-paste code
- Reusable components

### Configuration Duplication
- Duplicate config files
- Similar configurations
- Inconsistent naming
- Scattered settings

### Workflow Duplication
- Duplicate workflow steps
- Similar job definitions
- Repeated patterns
- Shared logic

### Documentation Duplication
- Duplicate documentation
- Similar content
- Overlapping topics
- Inconsistent information

### Tool Duplication
- Duplicate tools
- Similar scripts
- Redundant utilities
- Overlapping functionality

## Output Format

### Duplication Report
- Total duplication ratio
- Duplication by type
- High-impact duplications
- Consolidation opportunities
- Effort estimates

### Metrics
- Code duplication: [%]
- Config duplication: [%]
- Workflow duplication: [%]
- Documentation duplication: [%]
- Tool duplication: [%]

## Success Criteria
- [ ] All duplication identified
- [ ] Duplication ratio calculated
- [ ] Consolidation opportunities listed
- [ ] Effort estimates provided
