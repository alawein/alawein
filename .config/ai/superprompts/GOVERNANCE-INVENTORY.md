# Governance Inventory Prompt

## Objective
Catalog all governance policies, rules, and compliance requirements.

## Inventory Framework

### Policy Locations
- Where policies are stored
- Policy file locations
- Policy documentation
- Policy ownership

### Policy Types
- Code quality policies
- Security policies
- Compliance policies
- Deployment policies
- Release policies
- Documentation policies
- Testing policies
- Performance policies

### Policy Relationships
- Policy dependencies
- Policy conflicts
- Policy inheritance
- Policy composition

### Policy Enforcement
- Automated enforcement
- Manual enforcement
- Monitoring mechanisms
- Violation tracking

## Analysis

### Coverage Assessment
- Covered areas
- Gaps in coverage
- Overlapping policies
- Redundant policies

### Compliance Requirements
- Regulatory requirements
- Industry standards
- Internal standards
- Best practices

## Output Format

### Governance Inventory Report
- Policy catalog
- Policy locations
- Policy types
- Policy relationships
- Enforcement mechanisms
- Coverage gaps
- Compliance requirements

### Metrics
- Total policies: [count]
- Policy types: [count]
- Coverage: [%]
- Enforcement rate: [%]

## Success Criteria
- [ ] All policies cataloged
- [ ] Locations documented
- [ ] Types categorized
- [ ] Relationships mapped
- [ ] Coverage assessed
