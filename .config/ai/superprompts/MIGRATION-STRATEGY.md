# Migration Strategy Prompt

## Objective
Plan migration strategy for consolidation with minimal disruption.

## Strategy Framework

### Migration Phases
- Phase 1: Preparation
- Phase 2: Pilot
- Phase 3: Rollout
- Phase 4: Stabilization
- Phase 5: Optimization

### Migration Approach
- Big bang vs. gradual
- Parallel running
- Feature flags
- Compatibility layers

### Migration Validation
- Pre-migration testing
- Migration testing
- Post-migration validation
- Rollback testing

### Communication Plan
- Stakeholder notification
- Team training
- Documentation updates
- Support channels

## Strategy

### Backward Compatibility
- Compatibility layer
- Deprecation notices
- Migration guides
- Support period

### Data Migration
- Data mapping
- Data validation
- Data reconciliation
- Data backup

## Output Format

### Migration Strategy Document
- Phases and timeline
- Approach and rationale
- Validation strategy
- Communication plan
- Rollback procedures
- Success criteria

### Metrics
- Migration items: [count]
- Phases: [count]
- Timeline: [weeks]
- Risk level: [Low/Medium/High]

## Success Criteria
- [ ] Phases defined
- [ ] Approach selected
- [ ] Validation planned
- [ ] Communication ready
- [ ] Rollback procedures ready
