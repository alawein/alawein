# Policy Consolidation Prompt

## Objective
Analyze and consolidate scattered policies into unified framework.

## Consolidation Framework

### Policy Duplication Analysis
- Duplicate policies
- Similar policies
- Overlapping content
- Redundant requirements

### Policy Conflict Resolution
- Conflicting policies
- Contradictory requirements
- Priority determination
- Resolution strategy

### Policy Gap Identification
- Missing policies
- Uncovered areas
- Compliance gaps
- Best practice gaps

### Policy Standardization
- Consistent format
- Consistent naming
- Consistent structure
- Consistent documentation

## Analysis

### Policy Hierarchy
- Global policies
- Organization policies
- Project policies
- Team policies

### Policy Inheritance
- Base policies
- Specialized policies
- Override mechanisms
- Composition rules

## Output Format

### Policy Consolidation Report
- Duplication analysis
- Conflict resolution
- Gap identification
- Standardization plan
- Hierarchy design
- Inheritance rules

### Metrics
- Duplicate policies: [count]
- Policy conflicts: [count]
- Coverage gaps: [count]
- Standardization: [%]

## Success Criteria
- [ ] Duplicates identified
- [ ] Conflicts resolved
- [ ] Gaps identified
- [ ] Standardization planned
- [ ] Hierarchy designed
