# Rollback Procedures Prompt

## Objective
Define rollback procedures for each consolidation phase.

## Rollback Framework

### Rollback Triggers
- Critical failures
- Performance degradation
- Data loss
- Security issues
- Compliance violations

### Rollback Levels
- Step-level rollback
- Phase-level rollback
- System-level rollback
- Full rollback

### Rollback Procedures
- Rollback steps
- Rollback validation
- Rollback communication
- Post-rollback analysis

### Rollback Testing
- Rollback test plan
- Rollback test execution
- Rollback validation
- Rollback documentation

## Strategy

### Backup Strategy
- Pre-rollback backups
- Backup validation
- Backup retention
- Backup recovery

### Data Recovery
- Data backup locations
- Data recovery procedures
- Data validation
- Data reconciliation

## Output Format

### Rollback Procedures Document
- Rollback triggers
- Rollback levels
- Rollback procedures
- Rollback validation
- Communication plan
- Post-rollback analysis

### Metrics
- Rollback triggers: [count]
- Rollback levels: [count]
- Recovery time: [minutes]
- Data loss: [zero]

## Success Criteria
- [ ] Triggers defined
- [ ] Procedures documented
- [ ] Validation planned
- [ ] Communication ready
- [ ] Testing completed
