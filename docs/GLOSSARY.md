# Glossary

## A

**API** - Application Programming Interface. RESTful endpoints for platform
communication.

**Attributa** - Attribution platform for tracking marketing performance.

## E

**Edge Functions** - Supabase serverless functions for API endpoints.

## L

**LiveItIconic** - E-commerce platform for luxury merchandise.

**LLMWorks** - AI/ML tools and model management platform.

## M

**Monorepo** - Single repository containing multiple related projects.

## O

**Optilibria** - Optimization algorithms library with 31+ algorithms.

## P

**Platform** - User-facing application (REPZ, LiveItIconic, etc.).

**Package** - Shared library used across platforms.

## Q

**QMLab** - Quantum mechanics laboratory and simulation platform.

## R

**REPZ** - Performance analytics platform for athletes.

**RLS** - Row Level Security in PostgreSQL/Supabase.

## S

**SimCore** - Scientific computing and physics simulation platform.

**Supabase** - Backend-as-a-Service providing database, auth, and APIs.

## T

**Turbo** - Build system orchestrator for monorepo.

**TypeScript** - Typed superset of JavaScript.

## V

**Vite** - Fast build tool and development server.

**Vitest** - Testing framework built on Vite.

## W

**Workspace** - npm workspace for managing monorepo packages.
