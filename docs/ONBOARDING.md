# Team Onboarding Guide

Welcome to the Alawein Technologies team! This guide will help you get up to
speed quickly.

- [Quick Start Guide](./QUICK_START.md) completed
- Access to required tools and accounts

### Day 2-3: Codebase Familiarization

- [ ] Read [Testing Guide](./testing/TESTING-GUIDE.md)

- [ ] Pick up a "good first issue"
