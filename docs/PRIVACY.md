# Privacy Policy

Last Updated: 2025-12-09

This privacy policy describes how the Alawein platforms collect, use, and
protect user data.

## Overview

The Alawein monorepo contains multiple platforms, each with specific data
handling requirements:

| Platform     | Data Collected              | Purpose              |
| ------------ | --------------------------- | -------------------- |
| SimCore      | Simulation configs, results | Scientific computing |
| REPZ         | Workout data, progress      | Fitness tracking     |
| QMLab        | Experiment data             | Education            |
| LiveItIconic | Orders, preferences         | E-commerce           |
| LLMWorks     | Prompts, outputs            | AI tools             |
| Attributa    | Analytics events            | Attribution          |
| Portfolio    | Contact submissions         | Communication        |

## Data Collection

### Information We Collect

**Account Information**:

- Email address
- Display name
- Profile picture (optional)

**Usage Data**:

- Platform interactions
- Feature usage patterns
- Error reports

**Platform-Specific Data**:

- SimCore: Simulation parameters and results
- REPZ: Workout logs, exercise data, progress metrics
- QMLab: Experiment configurations
- LiveItIconic: Order history, shipping addresses
- LLMWorks: Prompts and generated content
- Attributa: Marketing attribution data

### How We Collect Data

- Direct input from users
- Automatic collection via analytics
- Third-party integrations (with consent)

## Data Usage

### Primary Uses

1. **Service Delivery**: Providing platform functionality
2. **Personalization**: Customizing user experience
3. **Analytics**: Understanding usage patterns
4. **Communication**: Sending relevant updates
5. **Security**: Protecting accounts and data

### We Do NOT

- Sell personal data to third parties
- Share data without consent
- Use data for unrelated purposes
- Retain data longer than necessary

## Data Storage

### Infrastructure

- **Database**: Supabase (PostgreSQL)
- **Hosting**: Vercel, Supabase
- **Region**: Data stored in US regions

### Security Measures

- Encryption at rest and in transit
- Row-level security in database
- Regular security audits
- Access logging and monitoring

## User Rights

### You Have the Right To

1. **Access**: Request a copy of your data
2. **Correction**: Update inaccurate information
3. **Deletion**: Request data removal
4. **Portability**: Export your data
5. **Objection**: Opt out of certain processing

### Exercising Your Rights

Contact: meshal@berkeley.edu

Include:

- Your account email
- Specific request
- Platform(s) involved

Response time: Within 30 days

## Data Retention

| Data Type            | Retention Period            |
| -------------------- | --------------------------- |
| Account data         | Until account deletion      |
| Usage analytics      | 2 years                     |
| Transaction records  | 7 years (legal requirement) |
| Support tickets      | 3 years                     |
| Deleted account data | 30 days (recovery period)   |

## Cookies and Tracking

### Types of Cookies

| Type       | Purpose                  | Duration |
| ---------- | ------------------------ | -------- |
| Essential  | Authentication, security | Session  |

- Browser settings can block cookies
- Some features may not work without cookies
- Analytics can be opted out

## Children's Privacy

- We do not knowingly collect data from children
- Parents can request deletion of child data

## International Users

- Data may be transferred to US servers
- We comply with applicable data protection laws
- EU users have GDPR rights

## Changes to This Policy

- We may update this policy periodically
- Significant changes will be notified
- Continued use implies acceptance
