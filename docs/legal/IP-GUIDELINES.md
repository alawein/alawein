### Trademark Usage Guidelines

**DO:**

- Use ™ symbol on first mention in documents
- Use the mark as an adjective (e.g., "Librex™ software")
- Maintain consistent capitalization
- Include trademark attribution in documents

**DO NOT:**

- Modify or abbreviate trademark names
- Use trademarks in domain names without permission
- Use trademarks in a way that implies endorsement
- Create derivative marks without approval

### Attribution Example

> Librex™ is a trademark of Alawein Technologies LLC. All rights reserved.
