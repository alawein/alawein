# Telemetry

> Observability and alerting for AI operations

**Category:** infrastructure

## Commands

### `npm run ai:telemetry`

Telemetry CLI

### `npm run ai:telemetry:status`

Show telemetry status

### `npm run ai:telemetry:alerts`

Show active alerts

## Exports

```typescript
import { telemetry } from 'tools/ai/telemetry';
```

## Examples

### Check telemetry status

```bash
npm run ai:telemetry:status
```
