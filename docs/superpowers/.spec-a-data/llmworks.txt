  with polished UI — arena debate mode, benchmark runner with progress bar, radar/bar

  On interaction, all "runs" produce random numbers after a simulated delay. A technically

  ("Runs evaluation and benchmark flows against configured providers") and the actual

README quality: Concise, well-written, correct voice. Claims "OpenAI and Anthropic

  given zero integration code exists. The stack section accurately lists the technology
  but omits that provider integrations are unimplemented. No badges, no live demo link
  state (just URL). README is not the problem — implementation gap is.

Key presentation gap: Any technical reviewer who clicks "Start Benchmark", waits, then
