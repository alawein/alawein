# Organizations

LLC-specific projects and business entities.

## Structure

| Organization | Description |
|--------------|-------------|
| `alawein-technologies-llc/` | Alawein Technologies LLC projects |
| `live-it-iconic-llc/` | Live It Iconic LLC projects |
| `repz-llc/` | Repz LLC projects |

## Relationship to Apps

- **`organizations/`** - Business entity containers with governance docs
- **`apps/`** - Deployable applications (may be owned by organizations)

## Adding a New Organization

1. Create directory: `organizations/<name>-llc/`
2. Add README.md with organization details
3. Add governance documents as needed
4. Link to apps in `apps/` directory
