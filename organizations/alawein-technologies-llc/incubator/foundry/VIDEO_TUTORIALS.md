## üé¨ How to Contribute Videos

### If You Want to Record

1. **Pick a video** from the list above
2. **Use the script outline** (customize as needed)
3. **Record with clear audio** (headset recommended)
4. **Add slides/demos** (shows are better than talks)
5. **Add captions** (YouTube auto-captions work)
6. **Submit link** to project

### If You Want to Write Scripts

1. **Pick a video** without a script
2. **Write detailed script** (conversational tone)
3. **Include timing** (e.g., [0:00-0:30] intro)
4. **Add talking points** (key ideas to hit)
5. **Submit for review**

### Video Standards

- ‚úÖ 4K or 1080p minimum
- ‚úÖ Clear, audible audio
- ‚úÖ Professional but friendly tone
- ‚úÖ Show demos, not slides
- ‚úÖ 5-12 min per video (shorter = better)
- ‚úÖ Captions included
- ‚úÖ Linked from this page + QUICK_REFERENCE.md
