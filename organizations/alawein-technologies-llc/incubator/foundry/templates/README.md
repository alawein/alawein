## Creating Custom Templates

### Email Template Format

```markdown
## [Template Name]

**Use when:** [When to send this]
**Subject line:** [Suggested subject line]
**Best time to send:** [Day and time]

### Copy

[Email body text - copy/paste ready]

---

### Customization Tips
- Replace [VARIABLE] with actual values
- Personalize if possible
- Keep short (under 200 words for mobile)
- Include clear CTA (call to action)

### Performance Tips
- A/B test different subject lines
- Measure open rate (2-week minimum)
- Track click-through rate
- Monitor unsubscribe rate
```

### Metrics Template Format

```markdown
## [Metric Name]

**What it measures:** [What does this metric tell you?]
**Target:** [What number should you aim for?]
**How to interpret:** [What does the number mean?]

### Formula

```
[Calculation method]
```

### Example

```
[Real example with numbers]
```

### Spreadsheet Setup

| Column | Formula | Notes |
|--------|---------|-------|
| Date | TODAY() | Auto-date |
| [Metric] | [Formula] | [Notes] |
```

## Best Practices

### Email Marketing

‚úÖ **Do:**
- Test subject lines
- Segment your list
- Track opens and clicks
- Follow up on replies
- Keep list clean

‚ùå **Don't:**
- Send too frequently
- Use misleading subject lines
- Forget unsubscribe link
- Ignore engagement metrics
- Send at random times

### Metrics Tracking

‚úÖ **Do:**
- Check metrics daily/weekly
- Set realistic targets
- Compare to benchmarks
- Document assumptions
- Review trends (not just numbers)

‚ùå **Don't:**
- Obsess over daily fluctuations
- Change metrics too often
- Use vanity metrics
- Ignore context
- Set targets without data
