## Overview

The Graph domain enables autonomous research discovery for graph-based optimization problems. HELIOS can:
- Solve graph coloring problems
- Optimize scheduling
- Design networks
- Solve routing problems

## Problem Types

### Graph Coloring
- Vertex coloring (minimum colors)
- Edge coloring
- List coloring variants

### Scheduling Problems
- Job shop scheduling
- Flow shop scheduling
- Open shop scheduling

### Routing Problems
- Traveling Salesman (TSP)
- Vehicle Routing (VRP)
- Arc Routing

### Network Design
- Minimum spanning tree variants
- Network flow optimization
- Connectivity optimization
