```bash

lsof -i :8501

streamlit run dashboard.py --logger.level=debug
```

### Server Won't Start
```bash
# Check port
lsof -i :8000

# Check logs
python server_enhanced.py 2>&1 | tee server.log

```

### API Timeout
```bash

# Restart dashboard
streamlit run dashboard.py
```

### Memory Issues
```bash
# Clear cache
curl -X POST http://localhost:8000/admin/clear

# Reduce cache size (in server_enhanced.py)
state.cache = Cache(max_size=500, default_ttl=120)
```
