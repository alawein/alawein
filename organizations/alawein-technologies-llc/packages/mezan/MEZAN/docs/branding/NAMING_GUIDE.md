# Naming Guide

Product vs Academic
- Product/code: “Mezan”
- Academic/papers: “MEZAN”

Solver Display vs Package
- Display (docs/UI): QAPFlow, WorkFlow, AllocFlow, GraphFlow, MetaFlow, DualFlow, EvoFlow.
- Package/imports: keep “libria” naming in code for brand continuity (e.g., `libria_meta`, `Librex.QAP_backend`).

Status Tags
- Use only: implemented, embedded, spec_only, baseline.

Examples
- “QAPFlow solver from the Libria Suite” (display)
- `from libria_meta.solvers.qapflow import QAPFlow` (code)
