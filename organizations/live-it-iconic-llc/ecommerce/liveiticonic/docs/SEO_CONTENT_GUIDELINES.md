## Page Optimization Guidelines

### Title Tags
**Formula**: `[Primary Keyword] - [Unique Value Prop] | Live It Iconic`

**Examples**:
- Homepage: "Premium Apparel Inspired by Automotive Excellence"
- Shop: "Shop Premium Apparel & Accessories"
- Product: "Black Premium T-Shirt - Precision-Cut Automotive Design"
- About: "About Live It Iconic - Automotive-Inspired Premium Apparel"

**Guidelines**:
- 50-60 characters (optimal for search results)
- Include primary keyword at the start
- Include brand name at the end
- Unique per page
- Action-oriented when possible

### Meta Descriptions
**Formula**: `[Hook] + [Key Benefit] + [CTA]`

**Examples**:
- Homepage: "Precision-cut luxury apparel with clean lines, durable materials, and motorsport-inspired design. Built for motion; refined for nights out. Shop Live It Iconic."
- Product: "Premium black t-shirt with automotive-inspired design and precision-cut fit. Premium quality apparel for discerning individuals. Shop now."
- About: "Discover our design philosophy blending motorsport discipline with luxury refinement. Learn how we cut, finish, and test every piece."

**Guidelines**:
- 150-160 characters (fits in most search results)
- Start with hook/benefit
- Include keywords naturally
- Include soft CTA (Shop, Learn, Discover)
- Unique per page
- Compelling and scannable

### H1 Tags
**Guidelines**:
- One H1 per page (required)
- Include primary keyword
- Descriptive and compelling
- 40-60 characters ideal
- Match page intent

**Examples**:
- Shop page H1: "Shop Our Collection"
- Product H1: "Premium Black T-Shirt"
- Category H1: "Apparel Collection"

### H2/H3 Structure
**Guidelines**:
- H2: Main section headers (include secondary keywords)
- H3: Subsections (supporting keywords)
- Never skip heading levels
- Create logical hierarchy
- Target 3-5 H2s per page

**Example Structure**:
```
H1: Premium Apparel Inspired by Automotive Excellence
  H2: Discover Our Collection
    H3: Premium T-Shirts
    H3: Luxury Hoodies
    H3: Signature Caps
  H2: Design Philosophy
    H3: Motorsport Discipline
    H3: Luxury Refinement
  H2: Quality & Durability
    H3: Premium Materials
    H3: Precision Manufacturing
```

## Content Recommendations

### Minimum Content Length
- Homepage: 500-800 words
- Category Pages: 300-500 words
- Product Pages: 200-400 words
- Blog Posts: 1,500-2,500 words
- About Pages: 800-1,200 words

### Content Best Practices
1. **Keyword Placement**:
   - Primary keyword in first 100 words
   - Keyword density: 1-2% (natural language)
   - Secondary keywords throughout
   - LSI keywords for semantic relevance

2. **Readability**:
   - 60-70 character line length
   - Short paragraphs (2-3 sentences max)
   - Use lists and bullets
   - Scannable formatting

3. **Internal Linking**:
   - Link from category to products (anchor text: product name)
   - Link from About to Shop
   - Link from products to related products
   - Use descriptive anchor text
   - 3-5 internal links per page

4. **Image Optimization**:
   - Descriptive filenames (not IMG_1234.jpg)
   - Alt text: 10-14 words, include keyword if natural
   - Compress images (tools: TinyPNG, ImageOptim)
   - Use WebP format with JPEG fallback
   - Proper dimensions (no stretching)

5. **Schema Markup**:
   - Organization schema (homepage)
   - Product schema (product pages)
   - BreadcrumbList schema (navigation)
   - LocalBusiness schema (if physical location)

## Image Alt Text Formula

`[Adjective] [Object] + [Context/Benefit]`

**Examples**:
- "Premium black t-shirt with precision-cut design for luxury lifestyle"
- "Live It Iconic apparel collection featuring motorsport-inspired pieces"
- "Precision-manufactured hoodie with automotive-inspired geometric patterns"
- "Signature cap combining motorsport aesthetics with refined styling"

**Guidelines**:
- 10-14 words (optimal for accessibility and SEO)
- First mention: brand name + product
- Subsequent mentions: can use pronouns
- Include keywords when natural
- Never keyword stuff
- Be descriptive for users with screen readers

## URL Structure Best Practices

**Recommended Format**: `/[category]/[product-name]`

**Examples**:
- ✓ `/apparel/premium-black-tshirt`
- ✓ `/hoodies/luxury-performance-hoodie`
- ✓ `/caps/iconic-baseball-cap`
- ✗ `/product.php?id=123`
- ✗ `/p/12345`

**Guidelines**:
- Use hyphens (not underscores)
- Lowercase only
- Include primary keyword
- Keep under 75 characters
- Descriptive, not cryptic
- Avoid session IDs and parameters
