# Contributing to REPZ

## Development Setup

```bash
npm install
npm run dev
```

## Code Standards

- **TypeScript**: All code must pass `npm run type-check`
- **Linting**: Run `npm run lint` before commits
- **Testing**: Maintain 80% coverage, run `npm run test:run`

## Commit Messages

Follow conventional commits format. See `.gitmessage` for template.

## Pull Requests

1. Create feature branch from `main`
2. Ensure all tests pass
3. Update documentation if needed
4. Request review from maintainers
