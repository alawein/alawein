# Attributa.dev Brand Guidelines

## Brand Identity

**Name**: Attributa.dev  
**Tagline**: Attribution Intelligence for Ethical AI  
**Mission**: Build AI that respects sources

## Core Values

- **Privacy-First**: Local processing, no unnecessary data transmission
- **Transparency**: Clear explanations and confidence levels for all findings
- **Ethical AI**: Promoting responsible AI development through attribution
- **Developer-Friendly**: Tools designed for developers and teams

## Visual Identity

### Typography
- **Primary Font**: Inter (web-safe sans-serif)
- **Monospace Font**: JetBrains Mono (for code and technical content)

### Color Palette
- **Primary**: Dark neutral (#222)
- **AI Detection Colors**:
  - High Suspicion: Red (#dc2626)
  - Medium Suspicion: Orange (#ea580c)  
  - Low Suspicion: Green (#16a34a)
- **Success**: Green (#16a34a)
- **Warning**: Orange (#ea580c)
- **Error**: Red (#dc2626)
- **Info**: Blue (#2563eb)

### Tone of Voice
- **Professional**: Clear, precise, authoritative
- **Educational**: Explain complex concepts simply
- **Cautious**: Emphasize limitations and appropriate use
- **Supportive**: Guide users toward ethical AI practices

## Key Messaging

### Primary Value Proposition
"Build AI that respects sources. End‑to‑end attribution, citation, and lineage for developers and teams."

### Core Features
1. Multi-signal AI detection (GLTR, DetectGPT, Watermark)
2. Citation validation and verification
3. Code security analysis
4. Privacy-first local processing
5. Explainable results with confidence levels

### Target Audience
- AI developers and researchers
- Academic institutions
- Technology teams implementing AI
- Organizations ensuring ethical AI practices

## Usage Guidelines

### Do
- Emphasize privacy and local processing
- Explain limitations and appropriate use cases
- Provide clear confidence levels
- Focus on attribution intelligence, not just detection
- Use technical accuracy in descriptions

### Avoid
- Claiming 100% accuracy in AI detection
- Suggesting tool should be sole basis for decisions
- Over-promising detection capabilities
- Ignoring privacy concerns
- Using fear-based messaging about AI
