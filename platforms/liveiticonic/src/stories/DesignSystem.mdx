# Live It Iconic Design System

Premium lifestyle merchandise brand for luxury automotive enthusiasts.

## Brand Philosophy

Live It Iconic represents the pinnacle of luxury lifestyle products. Our design system ensures consistency, accessibility, and premium presentation across all touchpoints.

## Core Design Principles

### 1. Luxury Through Simplicity
- Minimal, elegant interfaces
- Strategic use of white space
- Premium typography and imagery
- Refined color palette

### 2. Performance First
- Fast, responsive interactions
- Optimized for all devices
- Accessibility at the core
- Progressive enhancement

### 3. Accessible to All
- WCAG 2.2 Level AA compliance
- Clear focus indicators
- Screen reader friendly
- Keyboard navigation support

### 4. Consistency Across Platforms
- Unified component library
- Predictable patterns
- Reusable design tokens
- Single source of truth

**Primary Brand Colors:**
- **Gold** (#C1A060) - Primary accent, CTAs, premium elements
- **Charcoal** (#0B0B0C) - Dark backgrounds, headers
- **Cloud** (#E6E9EF) - Light backgrounds, light text
- **Ink** (#2D3142) - Body text, main content

**Semantic Colors:**
- **Success** (#10B981) - Positive actions, confirmations
- **Warning** (#F59E0B) - Caution, alerts

- Controls: 12px (buttons, cards, inputs)

## Accessibility

All components meet WCAG 2.2 Level AA standards:
- Color contrast ratios ‚â• 4.5:1 for text
- Keyboard navigation support
- Screen reader announcements
- Focus indicators (2px gold outline)
- Semantic HTML structure

## Best Practices

1. **Always use semantic HTML** - Use `<button>` for buttons, `<nav>` for navigation
2. **Provide alt text** - All images need descriptive alt text
3. **Test with keyboard** - Ensure full navigation without mouse
4. **Check color contrast** - Verify sufficient contrast for readability
5. **Use focus indicators** - Never remove focus outlines
6. **Test with screen readers** - Verify announcements and labels
7. **Provide error messages** - Clear, actionable error guidance
8. **Use ARIA sparingly** - Only when semantic HTML isn't available
