# REPZ Advanced Systems - User Journey Flows

## Overview

This document outlines the complete user journey flows for all four advanced systems in the REPZ platform.

## User Personas

### Primary Users
- **Performance Athletes**: Seeking advanced protocols for competitive edge
- **Longevity Enthusiasts**: Focused on health optimization and aging prevention  
- **Fitness Enthusiasts**: Comprehensive nutrition and training tracking
- **Medical Professionals**: Providing oversight for advanced protocols

## Journey Flow 1: PEDs Protocol Management

### User: Performance Athlete (Performance Tier)

#### Phase 1: Protocol Request
1. **Dashboard Access**: Navigate to `/protocols/peds`
2. **Protocol Browse**: Review available protocol types
3. **Medical Requirement**: System shows medical approval required
4. **Initial Consultation**: Schedule consultation through Medical Oversight
5. **Medical Review**: Doctor evaluates client health history
6. **Approval/Denial**: Medical clearance granted or denied

#### Phase 2: Active Protocol
1. **Protocol Activation**: Approved protocol becomes active
2. **Compound Setup**: Configure dosages and injection schedule
3. **Daily Tracking**: Log daily administration and effects
4. **Safety Monitoring**: Track side effects and vital signs
5. **Blood Work**: Schedule and upload regular lab results
6. **Progress Review**: Weekly check-ins with coach and doctor

#### Phase 3: Cycle Management
1. **Phase Transitions**: Prep → Active → PCT → Off
2. **Dosage Adjustments**: Based on response and side effects
3. **Emergency Protocols**: Automatic alerts for concerning metrics
4. **Cycle Completion**: Protocol closure and post-cycle therapy
5. **Next Cycle Planning**: Future protocol recommendations

### Key Decision Points
- Medical approval checkpoint (mandatory)
- Side effect severity assessment
- Blood work result evaluation
- Emergency intervention triggers

## Journey Flow 2: Comprehensive Nutrition Management

### User: Fitness Enthusiast (Adaptive Tier)

#### Phase 1: Food Database Setup
1. **Dashboard Access**: Navigate to `/nutrition/foods`
2. **Food Discovery**: Search 40,000+ food items database
3. **Personal Favorites**: Save frequently used foods
4. **Custom Additions**: Add personal food items with nutrition data
5. **Barcode Scanning**: Quick food entry via mobile integration

#### Phase 2: Recipe Creation
1. **Recipe Builder**: Navigate to `/nutrition/recipes`
2. **Ingredient Selection**: Choose foods from database
3. **Portion Calculation**: Set serving sizes and quantities
4. **Auto-Nutrition**: System calculates complete nutrition profile
5. **Recipe Testing**: Photo upload and preparation notes
6. **Recipe Sharing**: Community sharing (optional)

#### Phase 3: Meal Planning
1. **Plan Creation**: Navigate to `/nutrition/meals`
2. **Goal Setting**: Define caloric and macro targets
3. **Weekly Planning**: Assign recipes to meals and days
4. **Macro Optimization**: System suggests adjustments
5. **Grocery Generation**: Auto-create shopping lists
6. **Plan Activation**: Begin following meal plan

#### Phase 4: Daily Tracking
1. **Food Logging**: Track actual intake vs. planned
2. **Progress Monitoring**: Daily macro and calorie tracking
3. **Adjustment Alerts**: Notifications for off-track days
4. **Weekly Reviews**: Progress analysis and plan adjustments
5. **Long-term Trends**: Monthly and quarterly nutrition analysis

### Key Decision Points
- Macro target achievement
- Recipe complexity vs. time constraints
- Budget considerations for grocery lists
- Dietary restriction accommodations

## Journey Flow 3: Bioregulators Optimization

### User: Longevity Enthusiast (Longevity Tier)

#### Phase 1: Research and Selection
1. **Dashboard Access**: Navigate to `/protocols/bioregulators`
2. **Research Library**: Review scientific evidence and citations
3. **Protocol Comparison**: Compare different bioregulators
4. **Evidence Assessment**: Review research strength levels
5. **Medical Consultation**: Optional medical review
6. **Protocol Selection**: Choose optimal bioregulator

#### Phase 2: Protocol Implementation
1. **Dosage Configuration**: Set optimal timing and amounts
2. **Baseline Measurement**: Initial wellbeing assessment
3. **Protocol Start**: Begin bioregulator administration
4. **Daily Tracking**: Log effects across multiple dimensions
5. **Radar Visualization**: Track progress via radar charts
6. **Research Correlation**: Compare results to published studies

#### Phase 3: Optimization and Analysis
1. **Wellbeing Trends**: Analyze multi-dimensional progress
2. **Dosage Adjustments**: Optimize based on response
3. **Cycle Planning**: Plan on/off periods as needed
4. **Biomarker Integration**: Correlate with lab results
5. **Research Updates**: Stay current with new findings
6. **Protocol Refinement**: Continuous optimization

### Key Decision Points
- Evidence strength vs. potential benefits
- Dosage optimization points
- Cycle timing decisions
- Biomarker correlation insights

## Journey Flow 4: Medical Oversight Integration

### User: Medical Professional + Client (Performance+ Tier)

#### Phase 1: Professional Onboarding
1. **Doctor Registration**: Medical professional joins platform
2. **Credential Verification**: License and specialty verification
3. **Platform Approval**: Administrative review and approval
4. **Profile Setup**: Areas of expertise and consultation fees
5. **Client Matching**: Connect with appropriate clients

#### Phase 2: Client Consultation Process
1. **Consultation Request**: Client requests protocol review
2. **Scheduling**: Book consultation appointment
3. **Pre-Consultation**: Review client history and protocols
4. **Consultation Session**: Video/phone consultation
5. **Medical Assessment**: Comprehensive health evaluation
6. **Clearance Decision**: Approve, deny, or modify protocols

#### Phase 3: Ongoing Monitoring
1. **Regular Check-ins**: Scheduled follow-up consultations
2. **Progress Monitoring**: Review client tracking data
3. **Safety Alerts**: Respond to automated safety flags
4. **Protocol Adjustments**: Modify protocols based on response
5. **Emergency Response**: Handle adverse events
6. **Long-term Care**: Ongoing medical supervision

### Key Decision Points
- Medical clearance approval
- Safety alert responses
- Protocol modification needs
- Emergency intervention requirements
