# Tools

Development tools and utilities for the monorepo.

## Structure

| Folder | Purpose |
|--------|---------|
| `accessibility/` | WCAG accessibility auditing |
| `ai/` | AI tool integrations, MCP servers, compliance |
| `analytics/` | Analytics dashboard and insights |
| `cli/` | Command-line interfaces |
| `config/` | Configuration management |
| `devops/` | DevOps automation tools |
| `infrastructure/` | Backup, Docker, health checks |
| `marketplace/` | Marketplace integrations |
| `orchestration/` | Workflow orchestration (merged orchestrator + orchex) |
| `pattern-extractor/` | Code pattern extraction |
| `prompts/` | Prompt engineering tools |
| `recommendation-engine/` | AI recommendations |
| `scripts/` | Shell scripts |
| `security/` | Security scanning tools |
| `telemetry/` | Telemetry and metrics |
| `templates/` | Project templates |
| `utilities/` | Shared libraries and binaries |

## Usage

Most tools can be run via the CLI:

```bash
npx automation <command>
# or
python -m automation <command>
```
